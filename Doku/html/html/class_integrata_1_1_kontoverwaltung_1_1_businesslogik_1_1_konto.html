Detailed Description

Repräsentiert ein Bankkonto mit grundlegenden Funktionen wie Einzahlen und Auszahlen. Beinhaltet Kontoinhaber, Kontonummer, Dispositionsrahmen, Währung sowie eine einfache IBAN-Darstellung.
